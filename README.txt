# Sistema de Controle para Microrrede Híbrida

Projeto desenvolvido no curso técnico em eletrotécnica com foco em automação e gerenciamento energético utilizando Arduino.

## Objetivos
- Gerenciamento de fontes híbridas;
- Controle de carga e descarga de baterias;
- Priorização de cargas;
- Acionamento automático de relés;
- Simulação de microrrede inteligente.

## Tecnologias
- Arduino
- C++
- LCD I2C
- Relés
- Wokwi

## Estados do Sistema
- Normal
- Economia
- Emergência
